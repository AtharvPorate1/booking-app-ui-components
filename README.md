
# How to run locally

1. Clone the repository

```http
    npm install
```


```http
    npm start
```
##install json server globally
```http
    npm install json-server
```



```http
    json-server --watch hospital.json --port 3002 
```

```http
    json-server --watch db.json --port 3001
```
